Hentikan runtime hanya jika ada alasan operasional yang jelas, misalnya perawatan terencana atau penanganan masalah tertentu. Hindari menghentikan site saat pengguna sedang aktif kecuali benar-benar diperlukan.

Kalau Anda hanya butuh penyegaran layanan, `restart` biasanya lebih tepat daripada `stop`.
