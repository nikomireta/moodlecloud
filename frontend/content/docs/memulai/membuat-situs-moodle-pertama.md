Pembuatan site di Moodlepilot dibangun agar tim tidak perlu mulai dari server kosong. Fokus Anda adalah menyiapkan identitas site dan memilih kapasitas yang paling masuk akal.

## Data yang harus disiapkan

Sebelum membuka form [Buat Situs](/buat-situs), siapkan:

- nama site
- subdomain awal
- nama dan email admin utama
- paket yang sesuai kapasitas

Kalau masih ragu soal kapasitas, baca [panduan memilih paket dan memahami quota](/dokumentasi/pembuatan-situs-dan-domain/memilih-paket-dan-memahami-quota).

## Tahap pembuatan site

Di halaman pembuatan, Anda akan melewati tiga langkah:

1. Informasi site
2. Konfigurasi admin dan paket
3. Konfirmasi pembuatan

Setelah form terkirim, Anda akan melihat status provisioning. Site bisa langsung aktif atau tetap berada di status proses sampai semua langkah selesai.

## Setelah site aktif

Begitu status site menjadi aktif, lanjutkan dengan:

- membuka admin access
- meninjau status runtime
- memeriksa pengaturan backup
- mulai menyiapkan custom domain jika diperlukan

Panduan lanjutan tersedia di [operasional dan runtime](/dokumentasi/operasional-dan-runtime/memahami-status-runtime-dan-admin-access).
