Data laporan berasal dari plugin laporan Moodlepilot yang berjalan di tenant Moodle Anda. Plugin ini mengumpulkan event penting, tracking, dan snapshot ringkasan untuk dikirim ke backend Moodlepilot.

Kalau plugin belum terkoneksi atau belum mengirim snapshot, dashboard report tidak akan punya data yang cukup untuk ditampilkan.
