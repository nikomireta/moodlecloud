Backup otomatis berjalan mengikuti pengaturan backup site Anda. Setelah diaktifkan, pastikan Anda sesekali memeriksa status backup terakhir, ukuran arsip, dan retensi agar ritmenya tetap sehat.

Jika perlu perlindungan tambahan sebelum perubahan besar, buat juga backup manual.
